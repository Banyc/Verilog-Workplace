基本I/O：MUX
目标：简单I/O，熟悉ISE软件，熟悉Spartan-3板。
输入：
　开　关：自左至右表示从高到低8位二进制数。上/下对应：1/0。
　按　钮：自左至右对应数码管第12、23、34、41位显示两位十六进制。
输出：
　发光管：自左至右与开关对应。
　数码管：根据按钮选择显示开关所表示的十六进制。
